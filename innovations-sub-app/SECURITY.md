## Reporting a Vulnerability

We take the security of our ecosystem seriously. If you believe you have found a security vulnerability in the Innovana Immersive Web & AR/VR platform, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

Please send an email to **security@innovana.com** (placeholder) with the following details:

1.  **Type of Issue:** (e.g., XSS, SQL Injection, Remote Code Execution)
2.  **Full Path/URL:** Where the vulnerability can be found.
3.  **Proof of Concept:** Steps to reproduce the issue.
4.  **Impact:** What data or access is at risk.

### Response Timeline

*   We will acknowledge receipt of your report within 48 hours.
*   We will provide a status update every 5 business days.
*   We aim to fix critical vulnerabilities within 14 days.

Thank you for helping keep Innovana safe!
